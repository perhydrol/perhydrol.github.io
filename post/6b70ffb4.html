分类问题：逻辑斯谛回归与线性判别分析

分类

本文将专注于以下内容： 1. 为什么不使用线性回归模型 2. 什么是逻辑斯谛回归 - 单预测变量 - 多预测变量 3. 贝叶斯定理估计 - LDA （线性判别分析） 4. 灵敏度和特异度

为什么不使用线性回归模型

考虑《统计学习导论》中的例子：预测一个人是中风、服药过量、癫痫发作。如果使用线性回归模型，一个自然的假设是

\[Y= \begin{cases}1, & \text { 中风 } \\ 2, & \text { 服药过量 } \\ 3, & \text { 癫痫发作 }\end{cases}\]

然而，这会产生两个问题： 1. 隐含信息：三种情况的差距是一样的 2. 不同的编码顺序产生不同的模型

当然，如果只有两个响应变量自然，或者响应变量之间确实有一个程度顺序（温度的高、中、低），使用这种编码也是可以的。

当只有两个变量时，可以使用哑变量来进行响应变量的编码。接着进行线性回归拟合，产生一条直线，对于 Y>0.5 的分类为服药过量。然而，我们知道这条直线会有 [0,1] 之外的值，这些值难以解释。

\[Y= \begin{cases}0, & \text { 中风 } \\ 1, & \text { 服药过量 }\end{cases}\]

只有两个响应变量：逻辑斯蒂

有一类函数被成为 S 型曲线，而逻辑斯蒂模型就是这样的曲线。

\[p(X)=\frac{e^{\beta _0+\beta_1X}}{1+e^{\beta _0+\beta_1X}}\]

整理得出：

\[\ln{\frac{p(X)}{1-p(x)} } =\beta _0+\beta_1X\]

所有逻辑斯蒂回归可以视为一个关于 Ｘ 的线性模型
